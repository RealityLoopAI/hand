# 灵巧手操作指南

## 项目概述

这是一个基于ESP32的机器人手部控制系统的二次开发，通过MQTT协议接收控制指令，控制灵巧手实现抓烧杯和抓试管两个动作。

## 连接配置
- **WiFi名称**：`mqtt_server`
- **WiFi密码**：`12345678`
- **订阅主题**：`hand`
- **MQTT服务器地址**：`192.168.137.1`
- **MQTT服务器端口**：`1883`
- **MQTT服务器下载地址**：`https://www.emqx.com/zh/downloads-and-install/emqx-edge`

## MQTT控制命令

### 1. 动作控制命令

发送JSON格式消息到 `hand` 主题,名称为`hand`：

| 命令 | 功能描述 | 动作效果 |
|------|----------|----------|
| `beaker_setout` | 准备抓取烧杯 | 准备抓取烧杯的预备动作 |
| `beaker_grab` | 抓取烧杯 | 执行烧杯抓取动作 |
| `beaker_release` | 释放烧杯 | 释放烧杯动作 |
| `tube_setout` | 准备抓取试管 | 准备抓取试管的预备动作 |
| `tube_grab` | 抓取试管 | 执行试管抓取动作 |
| `tube_release` | 释放试管 | 释放试管动作 |
| `ok` | OK动作 | 执行OK动作 |
| `reset` | 复位 | 手部复位到初始位置 |


**使用示例：**
```json
{"hand": "beaker_setout"}    //控制手部准备抓取烧杯
```

## 故障排除

### 常见问题
- **动作无响应**：检查电源连接，确保舵机正常工作。
- **动作异常**：重新启动，自动归零校准，确保舵机回到初始位置。
- **WIFI连接失败**：检查WiFi名称和密码是否正确，确保ESP32与WiFi网络相连。
- **MQTT连接失败**：检查MQTT服务器地址和端口是否正确，以及服务器运行正常。
